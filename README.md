Official PyTorch code for inference of

**StyleStream: Real-Time Zero-Shot Voice Style Conversion**

## Overview

StyleStream is the first real-time zero-shot voice style (timbre, accent, emotion) conversion system that achieves state-of-the-art conversion performance with an end-to-end latency of ~1s. [paper](http://arxiv.org/abs/2602.20113) | [demo](https://berkeley-speech-group.github.io/StyleStream/)
