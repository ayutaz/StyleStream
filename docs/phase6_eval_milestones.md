## M6.1: 評価モデルの統合

| 項目 | 内容 |
|---|---|
| **タスク名** | 客観評価用の外部モデル群の統合と動作検証 |
| **推定作業量** | 3日 |
| **依存関係** | Python環境セットアップ、GPU環境 |

### 成果物
- 5つの評価モデルラッパー（統一 `BaseEvaluator` インターフェース）:
  - `whisper_evaluator.py`: Whisper-large-v3 → WER (%)
  - `resemblyzer_evaluator.py`: Resemblyzer → S-SIM
  - `accent_evaluator.py`: accent-id-commonaccent-ecapa → A-SIM
  - `emotion_evaluator.py`: emotion2vec → E-SIM
  - `utmos_evaluator.py`: UTMOS → MOS推定
- 各モデルの単体テスト、GPU消費量・推論速度ベンチマーク

### 受け入れ基準
- [x] 全5モデルが16kHz音声入力で正常動作
- [x] Resemblyzer: 同一話者ペアで類似度 > 0.85、異話者で < 0.5
- [x] 各SIMメトリクスは「変換音声 vs ターゲット音声」のコサイン類似度として計算

### 注意点
- **accent-id-commonaccent-ecapa**: SpeechBrain Hub のモデル。`EncoderClassifier` を使用
- **emotion2vec**: 複数バリアント存在。`emotion2vec_base` から開始
- **Resemblyzer**: L2正規化の有無等、コサイン類似度計算方法を確認
